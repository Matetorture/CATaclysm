Game Basics

Objective

Collect cats, build powerful decks, defeat all bosses, and upgrade your base to unlock new features.

Main Currency

Money ($) - Earned by defeating bosses. Used to unlock deck slots, upgrade base, and pull new cards.

Card System

Cards have Attack, Speed, Crit Rate, and Attack Type.

Level them up by collecting copies. They will automatically level up when you have enough copies.

Progression

Defeat all 10 bosses in a category to unlock the next category and base upgrade.

Tips & Strategy

  • Match attack types to boss weaknesses
  • Use modifiers strategically per slot
  • Clone high-rarity cards to save money
  • Speed up base upgrades with span cats
  • Unlock deck slots early for more power

Slot Modifiers

  • B (Bloody): Attack ×3
  • S (Shiny): Speed ÷3
  • L (Lucky): Crit ×3
  • R (Rainbow): Attack ×2, Speed ÷2, Crit ×1.5
  • N (Negative): Changes type to boss weakness

Combat & Boss System

Combat Mechanics

Cards automatically attack based on their Speed.

Critical hits deal 2× damage.

Type Advantage: Deal 2× damage when your attack type matches boss weakness.

Bosses

Defeat all 10 bosses to unlock next category.

The reward has a fixed base amount, but the final payout may randomly vary by up to ±30%.

Speed Up Upgrades

Place cats in upgrade slots to reduce build time using their Crit DPS:

New Time = Base Time ÷ Total Crit DPS

Remove or add cats anytime during upgrade.

Card Cloning

Clone cards to earn copies without pulling. Cloning time varies by rarity, reduced by clone speed multiplier.

Cards at max level (62 copies) cannot be cloned.
